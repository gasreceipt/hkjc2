# Legacy Python Prototype

This folder preserves the original Streamlit implementation while the app is rebuilt as a proper React/Vite front end.

It contains:

- `app.py` Streamlit UI
- `scraper.py` Python scraper wrapper and fallback data
- `node_scraper.js` HKJC Node API bridge
- `model.py` XGBoost model helper
- `data/` historical CSV files
- Streamlit and Python/Node dependency manifests

Keep this folder as backend reference material for the next pass.
